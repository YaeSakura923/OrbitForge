# OrbitForge 🛰️

**3D Satellite Tracking & Orbital Visualization Engine**

Built on a high-performance 3D map rendering engine with WebGL and Three.js, extended with real-time satellite tracking and orbital visualization capabilities.

## Features

- 🛰️ **Satellite Tracking** — Real-time orbit visualization and position tracking
- 🌍 **3D Globe Rendering** — High-performance WebGL-based 3D globe
- 🗺️ **Multi-source Data** — Support for various map data source formats
- 🎨 **Thematic Maps** — Dynamic switching and customization of map themes

## Getting Started

```bash
git clone https://github.com/YaeSakura923/OrbitForge.git
cd OrbitForge
pnpm install
pnpm start
```
